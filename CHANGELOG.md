# Changelog

All notable changes to ChatWithAnyone will be documented here.

## 0.1.0 - 2026-06-05

- Prepared the project for open-source maintenance.
- Added README, MIT license, contribution guide, code of conduct, security policy, privacy notes, issue templates, PR template, CI, support policy, release checklist, and maintainer playbook.
- Removed prompt-content debug logging to support the local-first privacy model.
- Fixed Swift compile issues around `ContentView` structure and Foundation Models stream snapshots.
- Added basic tests for persona/session JSON round-tripping and chat-message behavior.
