# Recommended GitHub Repository Settings

These settings improve discoverability and maintainer trust. They need to be set in GitHub because they are repository metadata, not files.

## Description

Local-first iOS conversational persona app using SwiftUI and Apple's Foundation Models.

## Topics

- swift
- swiftui
- ios
- foundation-models
- local-first
- privacy
- accessibility
- open-source

## Features

- Issues: enabled
- Discussions: enable once there are real testers or users
- Projects: optional
- Wiki: disabled unless actively maintained

## Branch Protection

When there is more than one contributor:

- Require pull requests before merging.
- Require CI to pass.
- Require conversation resolution.
- Protect `main` from force pushes.
